# Bienvenido a la Práctica 1 de IA

Esta documentación cubre el uso de los scripts de procesamiento de datos y visualización.

## Estructura del Proyecto
Los scripts principales se encuentran en la carpeta `python-scripts/`:
* `main.py`: Punto de entrada principal.
* `src/`: Lógica interna del procesador.

## Instalación
Para instalar las dependencias, ejecuta:
`pip install -r requirements.txt`
